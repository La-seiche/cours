L'intégration

Le HTML :

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

Le CSS :

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fonf, etc.

Le développement

Le Javascript :

Javascript est un langage de programmation de script employé pour rendre une page web interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramètres.

Le PHP :

Le PHP est un langage de programmation libre, utilisé pour généré des pages web dynamique, qui sont donc modifiés en fonction de certaines informations (heure, nom de l'utilisateur, formulaire rempli pat l'utilisateur, etc.)

MySQL :

MySQL est un système de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de création de compte) pour ensuite être ré-utiliseé au moment approprié (exemple : connexion à un site internet avec identifiant et mot de passe).
